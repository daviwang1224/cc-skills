# 背景知识

本文档提供工作背景和个人特征信息，帮助 LLM 更深入理解你的思维方式、决策偏好和工作场景。

**这些信息的作用**：
- 让 LLM 更懂你，理解你的思维方式和决策逻辑
- 作为分析和提问的背景知识，提供足够的上下文
- 激发 LLM 的深度推理能力，而非约束它的思考方式
